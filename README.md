# Sistema Escolástico

Proyecto básico de base de datos para un sistema escolástico.

## Tablas del sistema

El sistema cuenta con tres tablas principales:

1. Estudiantes
2. Materias
3. Notas

## Descripción

La base de datos permite registrar estudiantes, materias y las notas correspondientes de cada estudiante por materia.

## Tecnologías utilizadas

- MySQL
- GitHub
